# tmux Keybindings

設定: [modules/home/tmux.nix](../../modules/home/tmux.nix)

**Prefix キー**: `Ctrl+A` (デフォルトの `Ctrl+B` から変更済み)

以降の表で `prefix +` と書かれているキーは「Ctrl+A を押して離してから次のキー」を意味します。

## カスタム (このリポジトリで追加)

### ペイン

| キー | 動作 |
| --- | --- |
| `prefix + \|` | ペインを右に分割 (現在のパスを保持) |
| `prefix + -` | ペインを下に分割 (現在のパスを保持) |
| `prefix + h` | 左のペインへフォーカス |
| `prefix + j` | 下のペインへフォーカス |
| `prefix + k` | 上のペインへフォーカス |
| `prefix + l` | 右のペインへフォーカス |
| `prefix + H` | ペインを左に5列リサイズ (連打可) |
| `prefix + J` | ペインを下に5行リサイズ (連打可) |
| `prefix + K` | ペインを上に5行リサイズ (連打可) |
| `prefix + L` | ペインを右に5列リサイズ (連打可) |

### ウィンドウ

| キー | 動作 |
| --- | --- |
| `prefix + Tab` | 直前のウィンドウへ切替 |

### その他

| キー | 動作 |
| --- | --- |
| `prefix + r` | 設定 (`tmux.conf`) リロード |

### コピーモード (vi-like)

| キー | 動作 |
| --- | --- |
| `v` | 選択開始 |
| `y` | macOS クリップボード (`pbcopy`) にコピーしてコピーモード終了 |
| `Y` | 同上 |

## デフォルト (tmux 標準)

### セッション

| キー | 動作 |
| --- | --- |
| `prefix + d` | デタッチ |
| `prefix + s` | セッション一覧 |
| `prefix + $` | セッション名変更 |
| `prefix + (` / `)` | 前/次のセッション |

シェルから:
- `tmux new -s <name>` — 新規セッション
- `tmux ls` — セッション一覧
- `tmux a` — 直前のセッションにアタッチ
- `tmux a -t <name>` — 名前指定アタッチ

### ウィンドウ

| キー | 動作 |
| --- | --- |
| `prefix + c` | 新規ウィンドウ |
| `prefix + ,` | ウィンドウ名変更 |
| `prefix + n` | 次のウィンドウ |
| `prefix + p` | 前のウィンドウ |
| `prefix + 0`〜`prefix + 9` | ウィンドウ番号でジャンプ |
| `prefix + w` | ウィンドウ一覧 |
| `prefix + &` | ウィンドウ kill |
| `prefix + f` | ウィンドウ名で検索 |

### ペイン

| キー | 動作 |
| --- | --- |
| `prefix + z` | ペインズーム切替 (最大化) |
| `prefix + x` | ペイン kill |
| `prefix + q` | ペイン番号表示 (押している間) |
| `prefix + !` | ペインをウィンドウに昇格 |
| `prefix + Space` | レイアウト切替 |
| `prefix + {` / `}` | ペイン入れ替え (前/後) |

### コピーモード

| キー | 動作 |
| --- | --- |
| `prefix + [` | コピーモード起動 |
| `prefix + ]` | ペースト |
| コピーモード中 `q` | 終了 |
| コピーモード中 `/` `?` | 検索 (前方/後方) |
| コピーモード中 `n` `N` | 次/前の検索結果 |

### コマンド・ヘルプ

| キー | 動作 |
| --- | --- |
| `prefix + :` | tmux コマンドプロンプト |
| `prefix + ?` | キーバインド一覧 (実機で確認できる) |
| `prefix + t` | 時計表示 |

## よく使うシェルコマンド

```bash
tmux new -s work      # 新規セッション "work"
tmux ls               # セッション一覧
tmux a                # 直前のセッションにアタッチ
tmux a -t work        # "work" にアタッチ
tmux kill-session -t work
tmux kill-server      # 全セッション終了
```
