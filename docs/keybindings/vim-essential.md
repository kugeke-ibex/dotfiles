# Vim / Neovim 日常操作チートシート

出典: [neovim使いこなす（Zenn）](https://zenn.dev/yuuzan_kaibara/articles/e8d3630b677e68)

LazyVim のカスタム map は [nvim.md](nvim.md)（`keys-nvim` / `kn`）。本ページは **Vim 共通の日常操作**（サンプル: `█` = カーソル）。

| 記号 | 意味 |
|------|------|
| `█` | カーソル |
| `Esc` | ノーマルモードへ |
| `:` | コマンドモード |

---

## モード

| キー | 動作 | サンプル（前 → 後） |
|------|------|---------------------|
| `i` | カーソル**前**から入力 | `hel█lo` → インサート開始 |
| `a` | カーソル**後**から入力 | `hel█lo` → `hell█o` |
| `I` / `A` | 行頭 / 行末からインサート | — |
| `o` | **下**に空行 → インサート | `line2█` の下に空行が開く |
| `O` | **上**に空行 → インサート | `line2█` の上に空行が開く |
| `v` | ビジュアル（文字） | `v` → `w` → `w` で単語範囲を選択 |
| `V` | ビジュアル（行） | `V` → `jj` で複数行選択 |
| `Ctrl+v` | ビジュアル（ブロック） | — |
| `Esc` | ノーマルへ戻る | — |
| `:` | コマンド（`:w` など） | — |

**`o` の例（縦に見る）**

- 前: `line1` / `line2█` / `line3`
- `o` の後: `line1` / `line2` / `█`（空行・インサート）/ `line3`

---

## 削除系・置換

| キー | 動作 | サンプル（前 → 後） |
|------|------|---------------------|
| `x` | 1 文字削除 | `ca█t` → `ct█` |
| `X` | カーソル上 1 文字削除 | — |
| `dd` | **1 行**削除 | 下記 |
| `2dd` | **2 行**削除 | `a` / `b█` / `c` / `d` → `a` / `d█` |
| `dw` | 単語まで削除 | `hello beau█tiful world` → `hello █world` |
| `d$` / `D` | 行末まで削除 | `hello beau█tiful` → `hello █` |
| `d0` | 行頭まで削除 | — |
| `diw` | カーソル上の単語を削除 | — |
| `r{字}` | 1 文字置換 | `c█t` + `ra` → `a█t` |
| `R` | 置換モード（Esc まで上書き） | `old█text` → 入力で上書き |
| `s` | 1 文字消してインサート | — |
| `S` | 行全体消してインサート | — |
| `u` | 取り消し（Undo） | `dd` の直後に `u` → 行が戻る |
| `Ctrl+r` | やり直し（Redo） | **ノーマルモードのみ**（`Esc` してから）。`u` のあと → 再び削除 |
| `.` | **直前の変更を繰り返し** | `cw` で単語変更後、別の単語で `.` |

**`dd` の例**

- 前: `first` / `se█cond` / `third`
- `dd` の後: `first` / `th█ird`（`second` が消える）

**ビジュアル併用**

- 範囲選択 → `d` / `c` / `y` で、選択部分だけ削除・変更・コピー

**Redo が効かないとき**

- **インサート中**の `Ctrl+r` は Redo ではなく「レジスタから挿入」。先に `Esc` でノーマルへ
- Markdown のプレビュー切替は `<leader>um`（`Ctrl+r` は Redo 用にしない）

## コピー＆ペースト

| キー | 動作 | サンプル（前 → 後） |
|------|------|---------------------|
| `yy` | 1 行コピー（ヤンク） | 見た目は同じ。レジスタに行が入る |
| `Y` | 1 行コピー（`yy` 同義） | — |
| `yw` | 単語コピー | `one two█ three` → レジスタに `two` |
| `y$` | 行末までコピー | — |
| `p` | **下**にペースト | 下記 |
| `P` | **上**にペースト | 下記 |
| `"*y` / `"*p` | クリップボード（環境依存） | — |
| `"+y` / `"+p` | システムクリップボード | LazyVim では `y`/`p` が連携することが多い |

**`yy` → `p` の例**

- 状態: `alpha` / `bet█a` / `gamma` で `yy`（`beta` をコピー）
- `p` の後: `alpha` / `beta` / `bet█a` / `gamma`（コピー行が**下**に挿入）
- `P` なら: `alpha` / `bet█a` / `beta` / `gamma`（**上**に挿入）

**ビジュアル + `y` → `p`**

- `V` → `j` → `y` で 2 行ヤンク → `G` → `p` でファイル末尾に貼り付け

## 検索と置換

### 検索（ノーマル）

| キー | 動作 | サンプル |
|------|------|----------|
| `/text` + Enter | 前方検索・ハイライト | `/foo` → 次の `foo` へジャンプ |
| `?text` + Enter | 後方検索 | — |
| `n` | 次の一致 | 2 番目の `foo` 上で `n` → 3 番目へ |
| `N` | 前の一致 | 逆方向 |
| `*` / `#` | カーソル下の単語を検索 | `pri█ce` + `*` → 次の `price` へ |
| `:noh` | ハイライト解除 | `/foo` 後に実行 |

- 本 repo: ノーマルで `Esc` もハイライト解除（[nvim.md](nvim.md)）

### 置換（コマンド）

| コマンド | 動作 | サンプル |
|----------|------|----------|
| `:%s/旧/新/g` | **ファイル全体**置換 | `color=red` → `colour=red`（全行） |
| `:%s/旧/新/gc` | 確認しながら置換 | `y` で置換 / `n` でスキップ |
| `:s/旧/新/g` | **現在行**のみ | カーソル行だけ `ok` → `done` |
| `:10,20s/旧/新/g` | **行範囲**指定 | 10〜20 行目だけ |
| ビジュアル + `:s/...` | **選択範囲**のみ | 選んだ行だけ置換 |

**全体置換の例**

```text
前:  color=red / color=blue / text
:%s/color/colour/g
後:  colour=red / colour=blue / text
```

## インデント・行の整形

| キー | 動作 | サンプル（前 → 後） |
|------|------|---------------------|
| `>>` | 現在行を右インデント | `█return` → `  █return` |
| `<<` | 現在行を左インデント | 逆 |
| `>` / `<` + motion | 範囲をインデント | `>2j` など |
| ビジュアル + `>` / `<` | 選択行をインデント | `V` → `jj` → `>` |
| `o` / `O` | 空行挿入 → インサート | [モード](#モード) 参照 |
| `J` | **次行と連結** | `a=1█` + 次行 `b=2` → `a=1 b=2█` |
| `gq` + motion | 段落を折り返し | `gqip` で段落整形 |

**`>>` の例**

```text
前:  function main() { / █return 1 / }
後:  function main() { /   █return 1 / }   ← 1 段右へ
```

## 移動（よく使う）

| キー | 動作 | サンプル（前 → 後） |
|------|------|---------------------|
| `h` `j` `k` `l` | 左・下・上・右 | — |
| `w` / `b` | 次 / 前の単語先頭 | `two█ three` → `th█ree` / `█two three` |
| `e` | 単語末尾 | — |
| `0` / `^` | 行頭 / 最初の非空白 | — |
| `$` | 行末 | `hello█` |
| `gg` / `G` | ファイル先頭 / 末尾 | — |
| `42G` / `:42` | 42 行目へ | — |
| `%` | 対応括弧へ | `if (█)` の `(` 上で `)` へ |
| `Ctrl+f` / `Ctrl+b` | 1 画面 下 / 上 | — |
| `zz` | カーソル行を中央へ | 本 repo: `zz`→`z`→`z` で循環（[nvim.md](nvim.md)） |

- 本 repo: `j` / `k` は折り返し対応（`gj` / `gk`）

## ファイル・終了

| コマンド | 動作 |
|----------|------|
| `:w` | 保存 |
| `:wq` | 保存して終了 |
| `:q` | 終了（未保存時は確認） |
| `:q!` | 保存せず終了 |
| `:e path` | 別ファイルを開く |
| `:sp` / `:vs` | 水平 / 垂直分割 |

- 新規作成: `i` → 入力 → `Esc` → `:w memo.txt`

## シェル・コメント

| 操作 | 動作 | サンプル |
|------|------|----------|
| `:!cmd` | シェル実行 | `:!ls` |
| `:%!cmd` | 全行をコマンドに渡す | `banana/apple/cherry` → `:%!sort` → 五十音順 |
| `:!!` | 直前の `:!` を再実行 | — |
| ビジュアル + `gc` | コメントアウト（LazyVim） | 複数行 → `#` 付き |
| `gcc` | 行コメントトグル | プラグイン依存 |
