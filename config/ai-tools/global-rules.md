# Guidelines

This file is the user-level global instruction for AI coding agents
(Claude Code: `~/.claude/CLAUDE.md` / Codex: `~/.codex/AGENTS.md` 共用).
It is automatically loaded at the start of every session and applies to all projects.
Follow every rule defined below.

## Top-Level Rules

- To maximize efficiency, **if you need to execute multiple independent processes, invoke those tools concurrently, not sequentially**.
- **You must think exclusively in English**. However, you are required to **respond in Japanese**.
- To understand how to use a library, **always use the Contex7 MCP** to retrieve the latest information.

## Programming Rules

- Avoid hard-coding values unless absolutely necessary.
